JSim Public API

The JSim Application Programmer Interface (API) is a set of documentation appropriate for creating Java code that interacts in useful ways with the JSim application.

  • Creating JSim plugins ;
  • Creating Java-based MML Functions and Procedures ;
  • Programmatic interaction with the JSim computational engine (planned but not yet available).

The JSim Public API is preliminary and most methods remain undocumented. Nevertheless, we feel it may be somewhat useful in its current, incomplete form. Those who find this API documentation insufficient for their purposes, may wish to get the JSim Source code .
